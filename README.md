# react-inline-center

## Install

```
npm install react-inline-center
```

## Demo

<https://codesandbox.io/s/react-inline-center-hu81q>

## Usage

```js
import InlineCenter from 'react-inline-center'

render(
  <>
    <Button>
      <InlineCenter>
        <MyIcon />
      </InlineCenter>
      label.
    </Button>
    <p>
      <InlineCenter>
        <image />
      </InlineCenter>
      text.
    </p>
  </>
)
```
